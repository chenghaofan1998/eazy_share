## Suggested Full Description

`Eazy Share` helps you capture long-form web pages and turn them into images that are easier to read and share.

It is designed for content that does not work well as a plain link. Instead of asking readers to click through immediately, you can let them preview the article content directly as a long image or a set of split images.

Key features:

- capture long web articles from arbitrary websites
- crop left, right, top, and bottom bounds
- export one long image or split into 3, 4, 6, or 9 images
- choose export quality
- attach a QR code so readers can scan to open the original page
- switch UI language between English and Chinese

Typical use cases:

- sharing research or articles in chat groups
- turning long pages into image-first social posts
- removing distracting sidebars or headers from web content
- making content easier to preview before readers open the source page

## 中文详细描述建议

`Eazy Share` 用来把任意网站上的长文章、长页面转成更适合分享的图片。

它特别适合那些“不适合只发链接”的内容。相比直接丢一个链接，`Eazy Share` 让读者先看到文章本身，再决定是否继续阅读全文。

核心功能：

- 截取任意网站长文章
- 支持左右上下边界裁剪
- 导出单张长图，或自动拆分为 3、4、6、9 张图片
- 支持不同导出清晰度
- 可附加二维码，扫码返回原网页
- 支持中英文界面切换

典型场景：

- 在群聊或社群里分享文章内容
- 把长网页整理成适合社交平台发布的图片
- 裁掉无关侧边栏、页头、广告等干扰元素
- 让读者先快速预览内容，再决定是否打开原文
