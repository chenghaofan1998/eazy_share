# Chrome Web Store Assets and Listing Guide

## Goal

This document is designed for launch materials that are:

- clear to first-time users,
- attractive in the store listing,
- searchable by users looking for screenshot and article-sharing tools,
- aligned with what `Eazy Share` actually does.

本文件用于整理一套适合上架 Chrome Web Store 的素材和文案，目标是：

- 让第一次看到插件的用户立刻明白用途，
- 提升商店页吸引力，
- 覆盖更多用户会主动搜索的关键词，
- 同时确保文案和产品真实能力一致。

## Core Positioning

`Eazy Share` is not just a screenshot tool.

It is a browser extension for turning long web articles into images that are easier to preview, share, and revisit through QR back-links.

`Eazy Share` 不是普通截图工具。

它的核心定位是：把任意网页长文转成更适合预览、分享、传播和扫码回看的图片。

## Primary Search Intent to Cover

Use these concepts naturally in the title, subtitle, and description:

- long screenshot
- full page screenshot
- webpage screenshot
- article to image
- website to image
- split screenshot
- share article as image
- QR code article share
- capture long article
- screenshot tool for web pages

中文可自然覆盖这些表达：

- 长截图
- 网页长截图
- 全页截图
- 网页转图片
- 文章转图片
- 长文分享
- 分图导出
- 二维码回原文
- 网页裁剪截图

Do not stuff keywords unnaturally. Keep them readable and product-led.

不要生硬堆砌关键词，优先保证可读性和转化表达。

## Listing Strategy

The store page should answer these three questions immediately:

1. What is this extension for?
2. Why is it better than sending a link?
3. What can I do with it in one minute?

推荐文案逻辑：

1. 先讲它是干什么的，
2. 再讲为什么“发图”比“发链接”更有效，
3. 最后讲用户点安装后马上能做什么。

## Recommended Store Copy

### English Tagline

`Turn long web articles into clean, shareable images.`

### 中文标语

`把网页长文变成清晰、好分享的图片。`

### English Short Description

`Capture long web articles, crop distractions, export split images, and add a QR code back to the original page.`

### 中文短描述

`截取任意网页长文，裁掉干扰区域，导出分图，并附加二维码返回原文。`

### English Long Description

`Eazy Share` helps you capture long web pages and turn them into images that people can read before they ever click a link.

Instead of sharing a bare URL, you can share the content itself. Export a full long screenshot, split it into multiple images, crop away sidebars or headers, and add a QR code so readers can open the original page when they want the full context.

Best for:

- long articles that are hard to share as plain links
- research pages you want to send in chats or groups
- content posts for image-first social platforms
- web pages with distracting sidebars, menus, or extra UI

Key features:

- full-page capture for long web content
- custom left, right, top, and bottom cropping
- export as one long image or split into 3, 4, 6, or 9 images
- optional QR code footer linking back to the source page
- English and Chinese interface
- multiple export quality options

### 中文长描述

`Eazy Share` 用来把任意网站上的长文章、长页面，转换成更适合预览和传播的图片。

相比直接发一个链接，它更适合“先让读者看到内容，再决定是否阅读全文”的分享方式。你可以一键截取完整网页，也可以裁掉无关侧边栏、页头、目录或广告区域，再导出成长图或多张分图。同时，你还可以在图片底部附加二维码，让读者扫码回到原网页继续阅读。

适合这些场景：

- 长文章不适合只发链接
- 需要在聊天、社群、微信群、Slack、Discord 中快速分享网页内容
- 想把网页内容整理成适合社交平台传播的图片
- 页面上有干扰性的侧边栏、导航、广告或无关区域

核心功能：

- 任意网页长截图
- 左右上下边界裁剪
- 导出单张长图，或自动拆分为 3、4、6、9 张图片
- 可附加二维码返回原网页
- 支持中英文界面
- 支持不同导出清晰度

## Visual Direction

For screenshots and promotional graphics:

- keep the article content readable,
- avoid overly technical UI screenshots without context,
- show the exported result, not only the popup,
- use short, benefit-led titles,
- avoid cluttered annotations.

建议视觉方向：

- 让“内容图本身”成为主角，
- 不要只截插件弹窗，
- 每张图都要体现实际结果，
- 标题尽量短，强调收益，
- 不要堆太多箭头和说明框。

## Suggested Asset Folder Layout

- `assets/store/screenshots/`
- `assets/store/promotional/`
- `assets/store/icon/`

## Final Check Before Upload

- screenshot titles match real product behavior
- no mention of unsupported features
- no old `LongShot` or `Feishu` wording remains
- QR description matches local generation behavior
- English and Chinese copy are consistent
