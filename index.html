Eazy Share

Turn long web articles into clean, shareable images.

Eazy Share is a browser extension for capturing long web pages, cropping distractions, exporting split images, and attaching a QR code back to the original page.

Eazy Share 是一个把网页长文转换成可分享图片的浏览器扩展，支持裁剪无关区域、导出分图，并通过二维码返回原网页。

Privacy Policy GitHub Repository Star on GitHub
What it does

Full-page screenshot, long article capture, split export, and QR back-links.

适合场景

网页长截图、文章分享、裁掉侧边栏与广告、把长文整理成更适合传播的图片。
