Privacy Policy / 隐私政策

Effective date: March 1, 2026 / 生效日期：2026 年 3 月 1 日

Eazy Share is a browser extension for capturing long web pages and exporting them as shareable images.

Eazy Share 是一款浏览器扩展，用于截取长网页并导出为适合分享的图片。

1. What Eazy Share Does / Eazy Share 的用途

  • Capture long articles or long-form web pages
  • Crop left, right, top, and bottom bounds
  • Export one long image or multiple split images
  • Optionally attach a QR code that links back to the source page
  • 截取任意网站上的长文章或长页面
  • 自定义左右上下边界
  • 导出单张长图或多张分图
  • 可选附加跳转原网页的二维码

2. Data Processed by the Extension / 扩展处理的数据

  • The current webpage content selected by the user for capture
  • The source URL entered by the user for QR code generation
  • User settings such as language, output mode, quality, link options, and crop bounds
  • Temporary image data created during capture and split export
  • 用户主动选择截图的当前网页内容
  • 用户输入的、用于生成二维码的原网页链接
  • 语言、导出模式、清晰度、链接选项、裁剪边界等用户设置
  • 截图和分图导出过程中产生的临时图片数据

3. What We Do Not Collect / 我们不收集什么

  • No user accounts
  • No personal profile collection
  • No sale of user data
  • No analytics, ads, or tracking scripts
  • No upload of captured webpage content to developer servers
  • No sending QR source links to third-party QR services
  • 不创建用户账号
  • 不收集用户画像
  • 不出售用户数据
  • 不使用统计、广告或追踪脚本
  • 不将截图网页内容上传到开发者服务器
  • 不把二维码链接发送到第三方二维码服务

4. How Data Is Used / 数据如何被使用

  • Reading page size and scroll position
  • Capturing the visible area of a page
  • Stitching captures into one image
  • Splitting the long image into multiple images
  • Generating a QR code locally inside the extension
  • Saving exported files to the user's device
  • Remembering user preferences between sessions
  • 读取页面尺寸和滚动位置
  • 截取网页当前可见区域
  • 拼接成长图
  • 将长图切分为多张图片
  • 在扩展内部本地生成二维码
  • 将导出文件保存到用户设备
  • 记住用户上一次使用的配置

5. Storage and Retention / 存储与保留

Eazy Share uses Chrome extension local storage and temporary browser cache. Temporary capture session data is used for short-lived editing and export, not for cloud hosting.

Eazy Share 使用 Chrome 扩展本地存储和临时浏览器缓存。临时截图会话数据仅用于短时编辑和导出，并不作为云存储或长期托管使用。

6. Permissions / 权限说明

  • activeTab: run capture on the current page
  • scripting: inject scripts required for page measurement and capture flow
  • tabs: communicate with the active tab
  • downloads: save exported images locally
  • storage: save preferences and temporary capture session data
  • <all_urls>: support user-initiated capture on arbitrary websites
  • activeTab：用于在当前页面执行截图
  • scripting：用于注入页面测量和截图流程所需脚本
  • tabs：用于和当前标签页通信
  • downloads：用于保存导出图片
  • storage：用于保存配置和临时截图会话
  • <all_urls>：用于支持任意网站上的主动截图

7. Network Access / 网络访问

The extension works primarily on-device. QR codes are generated locally inside the extension and are not sent to a third-party QR API.

扩展的主要处理逻辑都在本地完成。二维码已在扩展内部本地生成，不再发送到第三方二维码接口。

8. Children's Privacy / 儿童隐私

Eazy Share is not directed to children and does not knowingly collect personal information from children.

Eazy Share 并非面向儿童产品，也不会主动收集儿童个人信息。

9. Policy Updates / 政策更新

This policy may be updated if the extension's data practices or scope change. The latest version should be published with the project repository or product listing.

如果扩展的数据处理方式或产品范围发生变化，本政策会相应更新。最新版本应随项目仓库或产品页面一起发布。

10. Contact / 联系方式

For privacy questions, please open an issue at github.com/chenghaofan1998/eazy_share/issues.

如对隐私问题有疑问，请在 GitHub 仓库 issue 中联系。
